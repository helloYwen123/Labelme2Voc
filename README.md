# labelme2voc

Convert labelme annotated dataset to pascal voc 2012 dataset 

# Requirements  

```shell
pip3 install Pillow
```

# Usage:

```shell
python3 labelme2voc.py ./samples/labelme ./out 
```

This tool will generate VOC dataset with borders
